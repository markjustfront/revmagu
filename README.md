# Mar Aguilar Serrats Portfolio

A modern portfolio website for **Mar Aguilar Serrats (MAGU)**, showcasing her work as a:

- 3D Animator
- Story Artist
- Storyteller
- Visual Development Artist

The website highlights animation projects, storytelling work, storyboards, concept art, and personal artwork in a clean and responsive format.

---

## ✨ Features

### Home
- Fullscreen video hero section
- Glassmorphism-inspired design
- Featured artist introduction
- Portfolio navigation cards

### Animation
- Demo reel showcase
- Animation exercises and projects
- Responsive video gallery

### Story-Telling
- Original stories and projects
- Work-in-progress indicators
- Project descriptions and artwork

### Storyboard
- Storyboard portfolio
- Visual storytelling projects
- Sequential art showcase

### Art Portfolio
- Character design
- Concept art
- Visual development
- Masonry-style gallery

### Contact
- Social media links
- Professional contact information

## 🚀 Running Locally

Simply clone the repository and open `index.html` in your browser.

```bash
git clone https://github.com/yourusername/magu-portfolio.git
cd magu-portfolio
```

Then open:

```text
index.html
```

No build process or dependencies are required.
